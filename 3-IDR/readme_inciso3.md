# 💡 Encender LED – NUCLEO-L053R8

Inciso 3 en **STM32CubeIDE** para encender el LED del pin **PA5** usando el botón **PC13** en la placa **NUCLEO-L053R8**.  
Al presionar el botón, el LED se enciende.

---

### ⚙️ Detalles
- **IDE:** STM32CubeIDE  
- **LED:** PA5  
- **Botón:** PC13  
- **Lenguaje:** C
